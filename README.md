#红鲸家居企业产品展示网站

##项目简介
为红鲸家居有限公司开发的企业产品展示网站，采用前后端分离架构。

##技术栈
| 层级 | 技术 |
|:---|:---|
| 前端 | Vue.js 2.x + Element UI + Axios |
| 后端 | Node.js + Express |
| 数据库 | MySQL 8.0 |

##功能模块
###前台展示
- 首页（轮播图 + 推荐产品）
- 产品中心（产品列表 + 关键词搜索）
- 产品详情（大图、价格、分类、参数）
- 新闻动态（新闻列表）
- 关于我们（公司简介）
- 联系我们（在线留言）

###后台管理
- 管理员登录认证
- 产品管理（添加 / 删除）
- 留言管理（查看）
