红鲸家居产品展示网站 - 运行说明
一、环境要求
Node.js v16+
MySQL 8.0
Vue CLI：npm install -g @vue/cli
二、数据库信息
数据库名：redwhale
用户名：root
密码：123456
三、导入数据库
bash
net start MySQL80
mysql -u root -p123456
CREATE DATABASE IF NOT EXISTS redwhale DEFAULT CHARACTER SET utf8mb4;
USE redwhale;
source redwhale.sql;
EXIT;
四、启动后端
bash
cd server
npm install
node app.js
看到“后端运行在 http://localhost:3001”即成功。
五、启动前端
bash
cd client
npm install
npm run serve
访问 http://localhost:8080。
六、后台登录
地址：http://localhost:8080/#/admin
账号：admin
密码：123456
七、常见问题
问题	解决方法
npm 命令无法识别  	重装 Node.js，勾选“Add to PATH”
npm install 下载慢	npm config set registry https://registry.npmmirror.com
数据库连接失败	        检查 MySQL 是否启动、db.js 密码是否为 123456
前端页面空白	                确认后端已启动在 3001 端口
图片不显示	                确认 public/images 下有 sofa.jpg 和 table.jpg
